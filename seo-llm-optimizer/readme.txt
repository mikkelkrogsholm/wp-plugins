Optimize your WordPress content for both search engines and LLM systems.

== Description ==

SEO & LLM Optimizer helps you prepare your WordPress content for the future of search. This plugin generates clean, semantic content suitable for AI training and retrieval while maintaining SEO best practices.

= Features =

* Convert post content to clean, LLM-friendly markdown
* Intelligent content chunking for optimal processing
* Frontend button for accessing optimized content
* Caching system for improved performance
* REST API for programmatic access
* Per-post optimization settings
* Global settings management

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/seo-llm-optimizer`
2. Run `composer install` in the plugin directory
3. Activate the plugin through the 'Plugins' screen in WordPress
4. Configure settings under Settings > LLM Optimizer

== Frequently Asked Questions ==

= What is LLM optimization? =

LLM optimization prepares your content for Large Language Models (like ChatGPT, Claude, etc.) by converting it to clean, structured markdown that's easy for AI systems to process and understand.

= Does this affect my site's SEO? =

No, the plugin doesn't modify your published content. It generates optimized versions on-demand that are separate from your regular post content.

= Can I customize the chunk size? =

Yes, you can set a global default chunk size in the settings, and override it per-post in the post editor meta box.
